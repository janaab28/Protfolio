Leveraging Graphical User Interfaces (GUI), the system offers a seamless and user-friendly experience for both cinema administrators and moviegoers. Features: User registration/login, payment options, responsive UI. See more about the Filmax project

Quality and Safety

The Quality and Safety Airline Company System is a robust software solution developed using Java to ensure the highest standards of quality and safety in airline operations. NABIH

The project focuses on enhancing e-learning for individuals with ADHD through computational techniques, aiming to create a personalized application. By leveraging AI algorithms, the app identifies users' predominant learning styles—auditory, visual, or kinesthetic—and generates tailored training plans accordingly. Developed using Figma. See more about the NABIH project
